What Is HTML?

Basic HTML

HTML stands for Hypertext Markup Language and was created by Tim Berners-lee. This is used along with CSS (Cascading Style Sheets) and Javascript to create web application. The current version of HTML is HTML5. You can begin an HTML page on your own by opening any text editor and following the steps below. Be sure to save the file with ".html" at the end of the name. Ex: index.html

Basic Structure

Template

To begin an HTML page you will need three different elements. These are html, head, and body. The html element is the container for everyting else. the head and body are both placed withing html but not within each other.

<html>
<head> Metadata (non-visible content) </head>
<body>

Visible page content

</body>
</html>

Elements

  • <html> </html>
  • <head> </head>
  • <body> </body>
  • <br />
  • <header> </header>

A page in HTML is made up of nested elements. These elements, also called "tags", are enclosed in brackets. This is shown in the list above.

In general tags must be opened and closed as in the first three items in this list. However, some elements like <br /> are empty or self-closing and so not need a seperate closing tag.

Adding Text

Paragraph

The way most text will be added to your page is with paragraph tags: <p> Your text here! </p>

Headings

You will want to add headings in your page to designate different sections of the page or new subjects. You have six headings at your disposal and they differ in size from <h1> (largest) to <h6> (smallest). The text "Adding Text" on this page is using the <h1> tag and the text "Headings" is using the <h2> element.

Other Elements

Links

You can add links to your page using the anchor element: <a> </a>. These will show up as blue text that turns purple when clicked on. You specify a destination for the link usging the attribute: href="example.html" and can open the page in a new tab with target="_blank"

New Lines

You can begin a new line by using the <br /> element. This will move the following text down a single line.

Viewing Your Website

Viewing locally

To view your new web page on your computer simply locate the file (saved with the .html extension!!) and double click on it. Your default web browser should pull up your page. You will likely want to continue on to a CSS tutorial to spice up the aesthetics of your page.

Viewing over the World Wide Web

If you want to view your website online you will need to hire a web hosting provider and register a domain name (these can often be done seperately or together). The hosting provider will have seperate documentation explaining how to use their particular service. You will also want to look into CSS to make your page stand out and be easier to use for your visitors!
